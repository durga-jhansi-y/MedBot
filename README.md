MedBot
======

This repository contains the MedBot project. It includes a frontend built with Vite and React.

Frontend
--------

Prerequisites
- Node.js 18+ (includes `npm`).

Quick start (npm)
1. Open a terminal in the repo root and change to the frontend folder:

```powershell
cd frontend
```

2. Install dependencies and start the dev server:

```powershell
npm install
npm run dev
```

Alternatives
- Using pnpm:

```powershell
pnpm install
pnpm dev
```

- Using yarn:

```powershell
yarn
yarn dev
```

Open: http://localhost:5173/ (Vite default)

Notes
- If `npm` is not on your PATH after installing Node, you can run it directly:

```powershell
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

- The dev script is defined in `frontend/package.json` (`"dev": "vite"`).
